how it works

  1. open a novel on readhive.org and copy the URL from your browser
  2. paste it below and click fetch info to load the novel details
  3. set your chapter range, then click download epub
  4. the Python server runs locally and saves the .epub to your Desktop
  5. AirDrop it to your iPhone — it'll open straight in Books ♡

first time setup

make sure you've run pip3 install -r NOVEL-TO-EPUB/requirements.txt once, then start the local server with: 

python3 NOVEL-TO-EPUB/readhive_server.py

keep that Terminal window open while using this page.
