how it works

  1. search a title below to check ReadHive first, and if there is no match, open Novel Updates to find translator sites
  2. paste a supported source url and click fetch info to load the novel details
  3. set your chapter range, then click download epub
  4. the Python server runs locally and saves the .epub to your Desktop
  5. if a translator site is unsupported, the page will tell you it cannot generate an epub yet

first time setup

make sure you've run pip3 install -r NOVEL-TO-EPUB/requirements.txt once, then start the local server with: 

cd "/path/to/your/READS" && python3 NOVEL-TO-EPUB/readhive_server.py

for safety, this page only shows a generic command unless the owner is signed in on this browser.

keep that Terminal window open while using this page.
